Install / Uninstall

Install

  • When you download an installer
    • Please run the installer.
  • When you download a .zip file
    • Please extract files from it, and put them in an arbitrary folder.

Uninstall

  • When you installed with an installer
    • Please uninstall in Control Panel > Add/Remove programs.
  • When you extracted a .zip file
    • You have only to delete all files.
