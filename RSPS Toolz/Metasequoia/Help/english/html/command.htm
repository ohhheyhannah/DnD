Command Panel

You can select various operations and settle options or view property in command panel.

There are six groups. If all groups are not shown in the screen, you can scroll them with dragging the area which is not included in any groups.

System

It imports and exports files and settles object property or material property. This is as same as the one on menu.

Edit

It cancels operation, selects, and hides faces. This is as same as the one on menu.

Command

It selects operations done on 3D picture. [Details]

Edit option

It selects options in commands like move, rotate, or scale.

X,Y,Z It makes editing towards the direction of each axis possible.
H,P,B (head, pitch, bank) can substitute in some commands.
L It edits on the local coordinates in the command like move.
W It edits on the world coordinates in the command like move.
S It edits on the screen coordinates in the command like move.
Pt It can select vertices.
Ln It can select lines.
Fc It can select faces.
Rc It can select with rectangle.
It particularly becomes useful when you release [Fc] option.
Rp It can select with rope (freehand).
It particularly becomes useful when you release [Fc] option.
CObj It can only select the current object.
Grid It enables makes a snap to the grid. Options of grid can be set in menu [View] [Grid].
Sym It does the bilateral symmetric editing.

View

It settles the property of perspective view and three-sided view.

Display It can independently be settled the display mode in the perspective view and the three-sided view.
You had better use Direct3D as long as there are not any problems in the view condition or the stability of behavior.
Software (soft)
It draws pictures with rendering only by CPU which does not use video card functions.
Direct3D (D3D)
It draws pictures with Direct3D. It can show with high speed as it uses the hardware function of video cards.
OpenGL (GL)
It draws pictures with OpenGL. It can show with high speed as it uses the hardware function of video cards.
YZ,ZX,XY It shows mesh of each plane on the perspective view.
Size and intervals of mesh can be selected in [View] [Grid].
Axis It shows the X/Y/Z axes of coordinates.
Mesh It shows mesh on three orthogonal views.

Lighting

Click or drag on the sphere to adjust the direction of the light.

+ It add a light. You can register up to three.
- It delete a light
< > It switch the light to change the direction and the color.
Backlight It reverses the direction the light front and back.
Color It changes a color of the light.
