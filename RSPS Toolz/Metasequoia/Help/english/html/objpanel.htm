Object Panel

"Object Panel" controls objects. You can select ON or OFF with menu [Panel] [Object Panel].

When you move the mouse to the right edge of the main window dragging the title, it will be joined to the main window. But you can make it independent if do this together with pressing the Ctrl key.

Color is shown with highlight in the Current object. ("obj1" is the current object in the left picture.)

When you click the name of the object pressing a keyboard, the following operation is performed.

Ctrl It selects or deselects on the list.
Shift It makes a range selection on the list.
Alt Vertices and faces in the object are selected.
Alt+Ctrl Vertices and faces in the object are deselected.

And you can change the order of objects or merge it with other objects.

Objects are shown when the eye mark in ON.
When you click the object dragging the Alt key, only that or all objects are shown.
Objects are fixed and cannot be edited when the key mark in ON.
When you click the object pressing the Alt key, only that object or all objects become to be able to be edited.
It shows the display color of vertices and lines of the object. You can change from [Property].
It selects whether to fold the display of children-objects in the object panel.
Child-objects are all shown when it is looking down and they are hided when it is looking right.
("obj4" is hided under "obj3" in the left picture.)

New It creates a new object.
When you click the button pressing the Ctrl key, a new object is inserted after the current selected object.
Clone It creates a cloned object of the current object.
When you click the button pressing the Ctrl key, new objects are inserted after the current selected object.
Delete It deletes the selected objects.
Property It sets up the objects.
< > It moves the hierarchy level of the object by one upward or downward.
Misc It sets up the visibility or the editing.
Show only current
It only shows the current object and does not show other objects.
Show all
It shows all objects.
Lock non-current
It makes only the current object to be edited and locks others to be edited.
Unlock all
It makes all objects to be able to be edited.
Merge all
It merges all objects into one.
Merge all visible
It merges the objects with display status into a single object. Objects which are not shown are untouched.
Merge selected
It merges the selected object in the list into a single object. Objects which are not shown are untouched.
Select vertices/faces
It selects all vertices and faces in the current object.
Unselect vertices/faces
It unselects all vertices and faces in the current object.
Show color
It selects whether to show the color of each object on the list in the object panel.
Show folder
It selects whether to show a hierarchical child objects. All objects are shown in the same line in spite of a parent-child relationship when you select OFF.
Edit also children
It selects whether the change of the status of visibility or locking is synchronized to child objects.
Set as default
It registers parameters in the selected object as default values.

Object Property

Object's name

It sets up the name, color, visibility, and the possibility of the editing.

The color selected in Configuration is applied when a cross mark is put on the colored section and the color selected here is applied when a cross mark is not shown. If you click the right side when a cross mark is not shown, it is put again.

Patch

It selects how to control faces of objects. Objects selected as patch can be changed to their patch into normal plane by menu [Object] [Freeze].

Flat
It shows normal planes. It can be used for both triangles and quadrangles.

Patch type 1
Basically, patch is applied to only quadrangles. It is applied to only triangles whose three sides are surrounded by quadrangles.
The edge of the unclosed patch is located away form vertices and lines. It becomes exactly same as "Patch type 2" for a complete closed patch.

Patch type 2
Basically, patch is applied to only quadrangles. It is applied to only triangles whose three sides are surrounded by quadrangles.
The edge of the unclosed patch is located near vertices and lines. It becomes exactly same as "Patch type 1" for a complete closed patch.

Catmull-Clark
It can be applied to both triangles and quadrangles.
When the [Reserve triangles] is ON, triangles in the original object will be subdividied into multiple triangles. Subdividing with reversing triangles can make a smooth patch around the border between a triangle and a quadrangle.
A Catmull-Clark subdivision makes different patches from Patch type 1 or 2. But it may give the same shape in other software because a Catmull-Clark subdivision has a compatibility with many other software that supports a Catmull-Clark subdivision.
(Caution: Please confirm a compatibility with an other software because some software make different patches from Metasequoia, and creased edges or open edges also make different patches.)

Mirroring

It selects the mirror shown in real time. You only need one side of the data of wire structure.

In [Separated], the copied face is created on the other side and each face is treated individually.

In [Joined], in addition to the face created on the other side, the face which can connect to the original face is created. When you put a limit on the joining distance, the face connecting both sides of the faces is not created if the faces are located in farther than the selected distance from the symmetrical boundary (XZ plane, when you make the mirror in the direction of X axis, for example).

Lathe

It selects lathe displayed in real time. You can make the rotated form of the line in the object around the selected axis.

Smoothing

It selects whether to make faces smooth. When menu [View] [Smoothing] is checked, objects which are selected on smoothing are displayed smoothly according to the angle of the faces.
