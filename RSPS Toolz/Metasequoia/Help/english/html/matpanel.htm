Material Panel

You can control color or mapping image on Material Panel.

You can select ON or OFF with menu [Panel] [Material Panel]. When you move the mouse to the left edge of the main window dragging the title, it will be joined to the main window. But you can make it independent if do this together with pressing the Ctrl key.

A "current material" is shown in the highlight color. ("mat1" is the current material in the left picture.)
When you click the name with pressing the Shift key, you can select vertices and faces to which the current material is assigned and you can unselect with pressing the Ctrl key.
When you drag the name of the material to other material, you can mix materials. And if you do this with pressing the Shift key, you can change the order of materials.

New It creates a new material.
Clone It creates a cloned object of the current object.
Delete It deletes the current object.
Property It sets up the material.
Others It deletes the unused material or loads and writes the file.
Delete unused materials
It deletes materials which are not attached to any faces.
Delte all
It deletes all materials.
Load from file
It loads materials from a file such as a mqm(Metasequoia Material) file.
Save to file
It saves to a supported file.
Set as default
It registers parameters in the selected material as default values.
Select vertices/faces
It selects vertices and faces that the selected material is attatched.
Unselect vertices/faces
It deselects vertices and faces that the selected material is attatched.

Property

Notice:
All the mapping is correctly shown on the globe of the name, but bump mapping is not shown on 3D picture.

Name

Name
It settles the name of the material and the shader also.
Shader
It selects how to show brightness towards light. 
Classic It is the shader used before Ver2.3.
Constant The base color is applied.
Lambert It changes brightness by diffusion light and ambient light.
Phong Specular highlight is added to Lambert.
Blinn Specular highlight is applied as same as Phong, but degree of reflection is different.
Vertex color
It uses the color selected [Vertex color] command as the base color for each vertex.
Double sided
Both front and back side of faces are shown in the 3D view.
Back side of faces are not always shown when the [Fr] button in the display options is pushed.

Property

Alpha
It selects how the object opaques. 0 stands for perfect transparency (nothing can be seen), and 1 stands for perfect opacity.
Diffuse
It selects to what degree it makes the lighted area lighter.
Emission
It selects to what degree it makes lighter irrespective of how the object is lighted.
Specular
It selects to what degree it makes the incident area of light lighter. It can be said in other words like speculum.
Specular power
It selects how wide or how small specular highlight is reflected.
Ambient
It selects to what degree it shows the color of the whole scene. It can be said in other words like environment light.
The color of the ambient light is settled with menu [View] [Document Information]. This example shows the case when the ambient light is bright red.

Mapping

It specifies a file name of a mapping image. When the file name is specified by a relative path, a file is searched in following order of folders;

  1. The folder (or the lower folder) at the .mqo file.
  2. The folder specified in the Configumration.

Please input a file name with an extension when you want to make a new file.

If you select pattern mapping and do not select alpha mapping, some file formats uses alpha channel of pattern mapping as alpha mapping.
