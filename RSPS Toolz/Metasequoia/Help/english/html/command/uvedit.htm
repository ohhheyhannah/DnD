UV edit

It operates the UV axis on the atlas.

Usually, please edit the UV axis manually after assigning UV value by using [Mapping] command.

Screen options

Unwrap
It shows a UV atlas.
CurObj
It shows a UV atlas of the current object only.
Material
It shows UV of only faces to which the current material is assigned.
Texture
It shows a UV atlas of material which has the same texture mapping as the current material.
Select
It shows a UV atlas of only selected faces.
Repeat
It repeatedly shows a mapping image in plural.
Grid
It shows a grid.
Snap
It sets up to snap grid when the UV vertex is moved.
Object
It shows 3D objects.
Select
It shows 'UV guide' like branches on vertices on which UV is selected.

Vertex / Face / Apex

It settles how to select on 3D model. It does not effect on the operation on the UV atlas.

Select / Move / Rotate / Scale

It settles how to operate on the UV atlas.

In [Move] mode only, you can select vertices on the UV atlas with rectangle and with rope when you press [Rect] or [Rope] in Edit option.

Expand

It selects continuous vertices from the UV vertex specified on the UV atlas.

Brush

It selects the traced UV vertices by dragging on the UV atlas.

The size of the selection range can be changed by dragging with the Alt key.

Rect

It selects UV vertices in the rectangle by dragging on the UV atlas.

Rope

It selects UV vertices in the freehand region by dragging on the UV atlas.

Magnet

It moves UV vertices continuously within the distance from the position of the UV vertex specified on the UV atlas.

New mapping

You can newly assign UV coordinate for the selected face by the following projection methods.

Plane mapping
Cylinder mapping
Sphere mapping
Apply
It does UV projection towards the selected faces under the current mapping condition.
Move
It moves the base point of projection mapping.
Rotate
It rotates projection mapping.
Scale
It scales up projection mapping.
Fit
It adjusts the scale-up ratio to the size just fits to the selected faces.
Property
It adjusts position, size, and angle of projection mapping with a numerical input.

Numerical input (U/V/OK)

It sets the coordinate of the selected UV vertex with a numerical input.

Normalize

It compulsory moves the position of UV vertices within the proper range when they are located outside the range of 0 - 1. It does not change a visible tension of mapping.

Unify

It joints the selected UV vertices.

It separates the selected UV vertices.

It moves the position of UV vertices like the position of polygon faces on 3D screen can be projected on the position of the UV atlas.

Align

It aligns the position of the selected UV vertices in a straight line vertically or horizontally. [Align] buttons automatically determines the direction to align whether horizontal or vertical. [U] button or [V] button enables to specify the direction manually.

Copy

It records the UV coordinate of the selected face. (See the following [Paste].)

Paste

It assigns the recorded UV coordinate with [Copy] to the currently selected face. The number of vertices of the source face and the destination face must be same.

SW Rotate

It replaces the UV value of the vertices in the polygon face one by one as rotating clockwise.

SW Invert

It replaces the UV value of the vertices in the polygon face as inverting to the reverse direction.

Atlas

It automatically assigns UV coordinates not to overlap UV faces on the atlas.
There is a defect that it is apt to become scraps in this version.

Export

It exports the atlas as the image file like BMP or JPEG.

Config

Length of guide
It selects the length of branches shown in Select in Object.
Grid
It selects how many to divide grid for mapping image.
Repeat
It selects how many to place mapping image in [Repeat] mode.
Snap to the center of grid
It snaps not on the crossing of grid but on the center of lattice of grid in [Snap] mode.
Move UV on the object
It can directly move UV value when you drag on the object.
Brightness of mapping image
It selects brightness of mapping image shown as the atlas. Lower brightness if it is hard to see the UV atlas when it is too bright.
