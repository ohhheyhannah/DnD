Vertex color

Vertex color is the color assigned to each vertex irrespective of the color of material. You can color vertices with [Vertex color] command. But you need to make vertex color available in material Panel. And it is not applicable to the object selected as patch.

When you click a vertex with pressing the shift key, its color changes to the currently selected color. When you do the same with pressing the Ctrl key, the color of the selected vertex is erased.

Vertex color is not supported in many file forms except MQO. Vertex color is mainly used for objects of low polygon in games.

RGB

It changes color only. Transparency does not change.

Alpha

It changes transparency only. Color does not change.

RGBA

It changes color and transparency at the same time.

Vertex

It changes the vertex color of clicked or dragged vertex.

Polygon

It changes the vertex color of each vertex within the clicked or dragged face.

Polygon

It changes the vertex color of the vertex within the clicked or dragged face.

Brush

It changes the vertex colors of the vertices that pass nearby when dragging.

Apply to selected face

It applys the specified color and transparency of vertices within the selected face.

Illuminate

It applies the illumination by shader to the vetex color of vertices within the face to which material with no vertex color is assigned.

Filter

It changes brightness, contrast, and color of the whole object.
