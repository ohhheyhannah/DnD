Magnet

It designates the selected vertex as the center and moves together with the neighboring vertices as if they are attracted by magnet.

This command is only available for the current object.

Tension icons

It sets up how magnet works.

Handle

It shows three arrows along with X/Y/Z axes, and you can move along with X/Y/Z axes with dragging.

Only selected

It moves affecting the selected vertices only.

Joined distance

If you check this item, it moves according to not spatial distance but distance of joined faces.

Set Range

It sets up how far magnet has an influence by dragging from the selected point.
