Local

It operates the local axis for each object.

[Local] command can change the shape based on a parent-child relationship and can move the vertices accurately along the local axis on the same plane. For example, if you use regular [rotate] command only, it becomes difficult to move an object along the axis after rotation because the rotation angle is reset once it is rotated. When you rotated the local axis, you can move an object along the local axis with [L] button in Edit option for [Move] command.

Shape

It selects whether to make it possible to move the position together with the vertex when a handle is operated. If you only want to change the location and the direction of the center, please do not check this.

Children

It selects whether to operate a child object together with a parent object.

Move

It sets up the center of the local axis with moving a handle.

Scaling

It scales up and down an object with operating a handle.

Rotate

It rotates an objects with operating a handle.

Centering

It sets the center position of the vertices of the current object as the basic point of the local axis of the object.

Property

It sets up the local axis with numerical input.
