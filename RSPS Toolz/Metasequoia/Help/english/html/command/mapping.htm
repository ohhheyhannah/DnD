Mapping

It assigns UV value for pasting mapping image to the selected face.

Mapping type

Buttons of shape icon represent flat, cylinder, sphere and UV mapping respectively.

Move / Rotate / Scale

It operates a basic shape for assigning mapping.

Invert U/V

It inverts each assigned direction of U and V.

Fit selected

It makes a basic shape of mapping fit to the selected face.

Fit CurObj

It makes a basic shape of mapping fit to the currently selected object.

Property

It shows dialog of a basic shape of mapping and sets up location, angle, and size in numerical values.
