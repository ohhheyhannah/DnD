Weight

A weight for a vertex or a crease for an edge makes a non-round object applying a patch.

A weight is only available for 'Spline type 1' or 'Spline type 2' patches. Weight can be set from -1.0 to 1.0. When you drag a mouse on the selected face, you can see the change of the shape in real time.

A weight is not applied for a Catmull-Clark patch.

A crease for an edge is available for a Catmull-Clark patch. After clicking an edge, a crease of the edge will be switched, and creased edges are highlighted. The following image is an example for applying creases to the edge on the top face.

Creases can be also applied by "Menu > Selected > Edge crease for Catmull-Clark".

Creases are not applied for 'Spline type 1' or 'Spline type 2' patches.
