Move

It moves selected vertices, lines and vertices.

Move

Rel (Relative)
The numeric input column indicates the offset of movement from the current vertex/line/face.
Abs (Absolute)
The numeric input column indicates the position.
Handle
It shows three arrows along with X/Y/Z axes, and you can move along with X/Y/Z axes with dragging.

When a right button is clicked while moving a vertex, the vertex is joined to the closest vertex. The dragged vertex is not joined and is arranged at the same position if the closest vertex is in the other object.
