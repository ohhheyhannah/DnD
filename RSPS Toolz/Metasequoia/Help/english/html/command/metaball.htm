Blob

It handles blobs which draw smooth faces by sticking or separating.

All operations for blobs like movement or polygonization are done with this command.

Blobs with positive density is shown in light orange, and blobs with negative density is shown in dark orange at default setting. (Color can be set up in Configuration.)

Arrage

Buttons of a shape icon newly arrange metaball.

Delete

It deletes clicked metaball.

Copy

It copies metaball clicked by left click.

Move

It moves metaball selected by left click.
If you want to use world axis system, it moves towards Y (or Z) with pressing Ctrl key.

Rotate

It rotates metaball selected by left click.

Scale

It scales up metaball selected by left click.
You can change the shape in equal proportion by dragging a mouse with pressing the shift key.

Propery

It sets up position, angle, size and density of metaball selected by left click.
When density is smaller than 0 (negative value), it can dent other metaball.

Polygonise

It polygonizes metaball as a new object. Regular editting can be done to a polygonized object.

You need to polygonise previously when you output it to the file excluding MQO.

Config

It sets up the lattice interval of metaball on the screen. The smaller interval becomes more precise, but it takes longer time to show on the screen.
