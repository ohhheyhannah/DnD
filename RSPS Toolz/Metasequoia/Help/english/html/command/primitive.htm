Primitive

It arranges nine types of primitives.

If you click [Property], the following dialog opens.

UV mapping

It automatically assigns UV value based on a shape.

Attach the current material

It assigns the currently selected material to the face of the created primitive.

Created as a new object

It adds a new object and registers faces to it every time a primitive is created.

Set a base position of the local axis

It designates the selected point when a primitive is created as the base point of the local axis of the obejct.
