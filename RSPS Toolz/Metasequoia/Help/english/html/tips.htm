Useful Tips

Shortcut

  • You can change the viewpoint to side, front, and upper with F1,2,3 keys and to the back direction for each with pressing the Shift key.
  • You can switch parallel projection and perspective projection with F4 key.
  • You can record the current viewpoint and edit option when you press F5-8 keys with pressing the Shift key and go back to the recorded data when you press F5-8 keys without pressing the Shift key.
  • If you press the S key as shortcut when the [Select] command is selected, the behavior of option changes. You can also use this in the same way in some other commands like [Wire].

Mouse operation

  • In the standard mode, the viewpoint rotates when you press the right button on 3D picture, but you can zoom the viewpoint when you press the left button with pressing the right button.
  • When you press [Pt], [Ln], and [Fc] buttons in edit option or view option at the top of 3D picture with a click on the right side, you can switch it in order.
  • If you click [Si] in view option on the top of 3D picture with pressing the right side of the mouse, you can select ON/OFF of simplified display of patch, mirror, lathe, and blob.
  • When you click the right side of the mouse with moving 1 vertex in [Move] command, you can join it with the vertex existing there.
  • If you drag a mouse up and down after pressing the upper or lower button on the right side of the value input column, you can change the value with how much you drag a mouse.

Menu

  • All objects are frozen if you select with pressing the Shift key in [Freeze].
  • The viewpoint goes back to the initial setting if you select with pressing the Shift key in [Set view].

About materials

  • When you set up materials, the new picture file is created if you input a file name which does not exist with an extension in the column of the file name of mapping image.

Ideas for faster display

  • Remove all checks in menu [View].
  • Press [Si] button (Show patches and blobs simply) and [Fr] button (Show only front lines) at the title in each view.
  • Make the number of dividing patches in the object selected with patches smaller while working.
  • Divide into as many objects as you can and hide objects which are unnecessary for editing.
  • Remove the checks of [Show precisely], [Show textures], and [Smoothing] if they are unnecessary.
  • Widen the lattice intervals of blobs.
  • Check which of Direct3D and OpenGL works faster or which property makes faster display like those in [Benchmark].
  • Lower [Max resolution of texture] in Configuration.
  • Make the size of the window smaller.

System

  • If you boot the PC after deleting metaseq.ini file, configuration goes back to initial setting.
  • When you don't use Direct 3D or Open GL, the PC boots a little faster if you remove a check in Configuration.
