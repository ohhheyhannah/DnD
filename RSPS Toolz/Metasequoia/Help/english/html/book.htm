Let's start with Metasequoia

Author: Daisuke Harada
Gijutsu-Hyohron Co., Ltd. (http://www.gihyo.co.jp/)
Price 1,980 JPY (+ tax)
Published in Jan., 2005

Order in Amazon.co.jp
Order in Rakuten books

This is a guide for beginners with which we can master 3DCG modeling with Metasequoia, the freeware version, which is said to be difficult, making pretty characters with full color. Many software and materials are contained in the CD-ROM. This is a convenient making kit for people who want to start.

Starting 3D-CG modeling with Metasequoia

Author: Kenji Kamoto
Kougakusha (http://www.kohgakusha.co.jp/)
Price 1,900 JPY (+ tax)
Published in Apr., 2005

Order in Amazon.co.jp
Order in Rakuten books

This book introduces, according to individual use, like mechanism as cars or robots, human body including pretty girl characters which instigated 3D-CG boom, and making of the creature which became familiar by movies. And instruction of function is also fully introduced. This book also explains how to use so-called subdivision surface which automatically changes from polygon shape to patches and mapping like texture, alpha channel, and bump.
