Paint panel

Paint Panel edits mapping image assigned to the current material. You can select ON or OFF in menu [Panel] [Paint panel]. And it is automatically shown when [Paint] command is selected.

In Paint panel, mapping image is shown as 2D image and you can draw in here. When you press the [Brush], [Palette], [Color] button, each panel is shown and it can select how to draw with brush or color.

Menu

File

Reload from file
It reloads the original image from the file making the editing result ineffective.
Save to file
It saves in the file.
Resize
It changes the size of the image.
Scroll
It scrolls the image. The value is a relative pixel to the original file image.
Effect
It apply special effects to the image.

Edit

Undo
It cancels the last operation of editing of the image.
Copy to Buffer
It temporally saves the image in buffer.
Paste from Buffer
It restores the image temporally saved in buffer and redraw the current image.
Copy to Clipboard
It transfers the image to the clipboard.
Paste from Clipboard
It transfers the image from the clipboard.

View

Realtime update
Even in the middle of drawing by left drag, it shows the result in 3D display. Response becomes quite slow.
