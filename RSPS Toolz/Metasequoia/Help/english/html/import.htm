LWO (LightWave3D)

When some UVs by multi-texture are assigned to one face, only one of them can be read.(*2)

COB (trueSpace)

The file saved in ASCII (text) format cannot be read.

As the hole opened by like Boolean cannot be handled, it will be closed. Please triangulate polygons on trueSpace beforehand in this case.

A polygon, if it exists, is automatically changed into a triangle or a quadrangle.(*1)

Metaballs with simple shape and polygonization are loaded as polygon objects.(*2)

OBJ (Wavefront)

A polygon, if it exists, is automatically changed into a triangle or a quadrangle.(*1)

A material library(MTL file) is loaded automatically if it exists in the same folder of OBJ file. It can only keep the name of material assigned to faces if there isn't the MTL file.(*2)

If there exists data written in some lines, it sometimes fails to load them.
