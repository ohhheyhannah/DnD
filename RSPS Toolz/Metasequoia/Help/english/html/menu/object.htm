Object

It works for the currently selected object.(You can select the current object on Object Panel.)

Create - Landscape (only shareware version)

It creates the rough mesh of landscape from an image file or fractal image.

Create - Create text (Plug-in)(only shareware version)

It creates strings using TrueType font.

Freeze

It entirely polygonizes the current object on which the patch or mirror is applied.

Shift the position

It arranges the current object at the position which is designated by the direction of each axis.

Join closed vertices

It joins the closed vertices in the same object.
It is mainly used to join vertices which seem to be close by appearance but they really are separated.

Reduce polygons

It reduces the vertices of the current object without changing appearance.

In this version, the result is not so nice and UV coordination is not kept. All faces become triangles.

Though the using availability is limited because the accuracy is not high, you had better reduce the number of vertices by 10 to 20 percent for finer finish after the polygonization of blobs.

Mesh smoothing (only shareware version)

It smoothes faces by increasing the number of faces.

It sometimes becomes roughly as more improvement is needed in this version.

Align faces

It aligns the direction of faces of the current object. But, whether they turn to the inside or the outside is not fixed, you may need to use [Invert]. And you cannot unify them well without closing all faces to one perfectly.

It is especially effective when a DXF or ROK file is read.

Delete overlapped faces

It deletes one of the faces in case two faces are located on the same position in the current object.

Unify faces

It deletes one side of faces which have both front and back sides in the current object. As which side will be deleted is unknown, you may need to use [Align faces].

Nothing is done with a double-sided face by the material property because there is one face only.

Attach current material to non-material faces

It attaches the current material to the non-material faces in the current object.

Wired (Plug-in) (only shareware version)

It makes all edges of faces to lines and deletes faces.

This may not be practical yet as this is only sample Plug-in.

Delete lines (Plug-in) (only shareware version)

It deletes all lines in the object.

Boolean (Plug-in) (only shareware version)

It picks out or makes up the only overlapping part of two objects using boolean operation. Please refer to Help in Plug-in for further details.

This Plug-in is developed by Mr.Bomber, and thanks his kindness.

Anguwrap (Plug-in) (only shareware version)

It generates an unwrapped UV mapping so that the selected lines become borders.

UV unwrapping is applied for all faces in the current object. And UV unwrapping will be failed when no line is selected.
