View

Settings for displays

Document Information

It shows document information and sets up the light source.

Set View

It can numerically input the direction of the viewpoint and initialize the viewpoint.

When you choose this menu with pushing the shift key, the viewpoint is initialized without showing the dialog.

Grid

It sets up the intervals of grids, the applicable axes, and intervals of plane meshes.

Set background images

It sets up the image file arranged in each three orthographic views.

You can set up the arranging location and size of background image with [BkImg] command..

Show precisely

It precisely shows the effect of the light source performed in [Document Information]. It delays a little to show.

Show textures

It shows the texture mapping.

Smooth shading

It can show the surface of the object smoothly. It delays a little to show.

It is not available for a patch (type1/2) and blob without polygonization.

Show only faces in a current object

It shows only faces in a current object.

Show lines of uncurrent objects

It specifies how to show vertices and lines of objects excluding a current object.

Normal
It shows vertices and lines with the assigned color to the object.
Dark
It shows vertices and lines with the color specified in 'Dark' in Configuration.
Hidden
It hides vertices and lines.

Show selected parts only

It shows the selected vertices and lines only, and hides the unselected part.

Lines overwrite faces

It always shows vertices and lines overwrite faces without considering for the z-order.
