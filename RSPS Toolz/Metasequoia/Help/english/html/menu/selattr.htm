File   Edit   Attribute   Selected   Object   View   Panel   Help

Attribute

It changes attribute of the selected part.

Hide selected faces

It temporarily hides the selected faces which are obstacle to work.

Hide unselected faces

It temporarily hides the unselected faces which are unrelated to work.

Show hidden faces

It shows the hidden faces by using [Hide selected faces] or [Hide unselected faces] again.

Fix selected faces and vertices

It fixes the position of the selected faces and vertices with [Move] command and so on.

Unfix selected faces and vertices

It unfixes the lock of [Fix selected faces and vertices] and makes it possible to change the position.

Fix selection

It fixes the selection for avoiding to release a selection with [Select], [Move] command and so on. Addition of the selection is possible.

It is not effective on the operation on Menu as [Unselect all].

Store selection

It stores the current selected part.

Restore selection

It selects the stored part by [Restore selection] all over again.

Set center of rotation

It sets the viewpoint to rotate around the selected area. When you click this command without any selection, the center of rotation is set to the origin.

It does not make sense when the center of rotation is set to except 'Fix' in Configuration.

Fit to view

It shows the selected area to be viewed on the whole screen.

If you click this command without any selection, the entire object is viewed on the whole screen.
