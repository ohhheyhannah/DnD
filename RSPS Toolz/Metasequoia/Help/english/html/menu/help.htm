Help

Index

It opens the top page of Help.

About

It shows the version or the copyright of Metasequoia.

About Plug-ins (only shareware version)

It shows the information on Plug-ins and installs the new Plug-ins.

Error Window

It shows the error message in case the illegal Plug-ins is read or some error occures.

Benchmark (only shareware version)

It measures the number of drawing sheets per second.

It can confirm which set-up makes the faster view. As the measuring is not so accurate, it is not so effective to use for the comparison with that of other hardware.

Homepage

It shows the maker's homepage by linking to Internet.

Registration (only shareware version)

It inputs serial ID and password necessary for removing restrictions on operation.
