File   Edit   Attribute   Selected   Object   View   Panel   Help

Selected

It works for the currently selected vertices, lines, and  faces.

Cut

It deletes the selected vertices, edges, and faces after copying them to the clipboard.

Copy

It copies the selected vertices, edges, and faces to the clipboard.

Paste

The selected vertices, edges, and faces that are on the clipboard can be copied on the new object.

Delete

It deletes the selected vertices, edges, and faces.

Invert

It turns over the selected faces.

Double sided face

It duplicates the selected faces to be seen from the front and the back.

Set material to faces

The current material is assigned to the currently selected faces.

Extrude faces

It extrudes the selected faces to make them stand out.

It functions the same as [Extrude] command, but this can numerically set up the extruding amount.

Mirror

It creates the inverted faces of the selected faces putting either of YZ/ZX/XY face between them.

As the vertices are not joint with the existing ones, use [Join closed vertices] if you need.

Faces to the new object

It creates a new object and moves the selected faces on it.

Merge two triangles

It merges the two selected triangle faces to make a quadrangle.

You can do the same work with [Merge two triangles into a quadrangle] in [Wire] command.

Triangulate

It divides the selected quadrangle to make two triangle faces.

You can do the same work with [Merge two triangles into a quadrangle] in [Wire] command.

Symmetry

It moves the selected two vertices to the symmetric position.

Join vertices

It joins the more than two selected vertices as one vertex. Vertices have to be in the same object.

Align vertices

It arranges the selected vertices to make a lines in the direction of each X/Y/Z axis.

Flatten

It moves vertices to flatten uneven parts or stand them out.

Round

It makes the round shape by erasing the selected lines. The shape changes in real time when you change a numerical value.

Erase vertex color (Plug-in)

It erases the color assigned to the selected vertices.

Separate faces (Plug-in)

It separates the selected faces and keeps the vertices independent from those of other faces.
