File

File input/output and Configuration

New

It cancels the currently edited document and draws a new document.

It confirms whether to save the edited document before going to a new document.

Open

It loads documents and objects from files. The currently edited document is cancelled.

 [Details].

Save

It saves the currently edited document. Input of the file name is requested if it is not given to the new file.

Save with counter increment

It is only shown when "Increase a file No. manually" is ON in Configuration. It saves the file as the new file with adding a file No.

Save As

It allows you to save a new copy of the current document, with a new file name. The original document is not changed.

[Details]

Insert

It loads objects from a file and inserts them to the current document.

Rendering

It exports an image shown on the perspective view as an image file such as BMP or JPEG with higher quality.

Export to DOGA-L

It registers parts directly to DOGA-L series developed by ProjectTeam DoGA.

Configuration

It sets up various operations.

[Details]

Basic Mode

It only shows the minimum needed menu and commands and hides commands which are not so important.

Please uncheck "Basic Mode" when you have not mastered basic operation.

Latest Files

It loads the latest files.

Quit

It exits Metasequoia. Whether to save the file is confirmed when it has not saved.
