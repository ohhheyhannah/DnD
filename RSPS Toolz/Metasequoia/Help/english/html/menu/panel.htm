Panel

It sets up showing or hiding of each panel.

Paint Panel

It shows the panel for editing mapping images.
[Paint Panel Details]

Script Editor (Only shareware version)

It shows the display for editing Python script.
[Script Editor Details]

Object Panel

It selects the current object, sets up visibility and editing, and shows the panel for setting up details on objects.
[Object Panel Details]

Material Panel

It selects the current material and shows the panel for setting up details on material.
[Material Panel Details]

Edit Panel

It can operate the selected part on each axis without using [Move] [Scale] [Rotate] command.

You can scale up and down in the equal ratio with dragging on an icon.

The picture rotates by 15 degrees at a time while you press the shift key.

Command Panel

It switches showing or hiding of Command Panel on the left side (or the right side) of the screen.
