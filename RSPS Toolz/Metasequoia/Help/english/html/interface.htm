Interface

Notice: Freeware version LE does not contain three orthogonal views. It contains only a perspective view.

The file name of the currently edited document is shown on Title bar.

Menu can conduct various operations.

The guide to the mouse operation is shown on Status bar on occasion demands. See here when you do not understand how to operate. And the number of the currently selected vertices (V) and faces (F) and ON/OFF of 'Selection Lock' is being shown at the right edge.

There are six groups of [System], [Edit], [Command], [Edit option], [View], and [Lighting] in Command panel, and you can conduct various operations in [Command]. If the groups cannot be settled in the window, it scrolls when you drag the area other than the groups. The Sub panel suitable for each command is opened when you click a command button.

3 views of Top view, Front view, and Side view whose angles are fixed can be said as Three orthogonal views. And if you put it together with Perspective view whose angle is freely switched, it can be called 3D views. Objects are shown in the method of any of [Software], [Direct3D], and [OpenGL] in each view. The method for display is switched with a button in the view group on Command panel.
