FAQ - Preview

In preview by Direct3D or OpenGL, many problems arise caused by like a bug or compatibility of the video card driver. As reported problems are also mentioned on the home page, please see them together with the following points.

It is recommend to obtain the latest driver from the maker of video cards or chips and use it to avoid problems.

Q1. I cannot use Direct3D preview at all.

A1-1. DirectX 9.0 runtime has not been installed
Please get and install DirectX 9 or later from the Microsoft DirectX site it if you use WindowsXP(no SP or SP1).
A1-2. Unsupport hardware rendering
Some old PCs do not support a hardware rendering. And some virtual environments like as VirtualPC or remote desktop do not support Direct3D. Please us other view mode in these cases because the Direct3D display mode is not supported.
A1-3. Bugs of the display driver
Please get and install the latest display driver.

Q2. I cannot use OpenGL preview at all

A2-1. Bugs of the display driver
Please get and install the latest display driver. If you still cannot solve the problem, reduce the level of hardware accelerator and switch off the accelerator of OpenGL. (How to change accelerator)

Q3. Application error or blue screen is occurred when Direct3D is used

A3-1. Bugs of the display driver
Please get and install the latest display driver.
A3-2. Problem for hardware
It may be caused by problem for the hardware such as physical failure, thermal runaway, and so on.

Q4. Texture is not shown

A4-1. Video memory is insufficient.
As video memory (VRAM) is limited in Direct3D or OpenGL mode, it cannot be shown when available capacity becomes insufficient.
You can take measures like:
  • Make the size of the picture for texture smaller.
  • Reduce the resolution or the number of colors of the picture.
  • Change to the video card with more video memory.
  • Reduce the value in [Max resolution of texture] in [Preview] page of menu [File] [Configuration].
But the setting itself of mapping image is effective even if it is not shown, so you do not have to worry about that according to the situation.
A4-2. Image file for texture is not put in the appropriate folder.
You need to put image folder in the folder selected in [Folder of mapping bitmaps] in menu [File] [Configuration].
A4-3. [Texture] is not checked.
Texture is not shown if there is not a check in menu [View] [Texture].
A4-4. Bugs of the display driver
Same as A1-3.

Q5. Selection of alpha mapping in Material Property cannot work on 3D screen.

A5-1. Video memory is insufficient.
Same as A4-1.
A5-2. Image file for texture is not put in the appropriate folder.
Same as A4-2.
A5-3. View Property is not checked.
Please check in menu [View] [Show precisely] and [Show textures].

Q6. Overlapped semi-transparent polygon is not shown correctly.

A6. It's a specification
As data processing cost becomes very large in order to show front and back of semi-transparent polygon. Semi-transparent polygon is not always shown correctly because high speed is regarded as more important even if it is incorrect. You can make more accurate the following:
  • Separate semi-transparent polygon into another object
  • Replace the object at the back with the forward in object panel

Q7. Selection of bump mapping in Material Property cannot work on 3D screen.

A7. It's a specification
Bump mapping cannot be shown in any view mode in this version. It is not reflected on the result in [Rendering]. It will be possible to be supported in future.

Q8. Background image is not shown.

A8. [Img] button is not selected.
Press the [Img] button at the title of each view.

Q9. How to save preview pictures in the file?

A9-1. Copy with the [Print Screen] key.
Capture the whole screen with the [Print Screen] key in the function of Windows and [Paste] from the clipboard with other picture editing software.
A9-2. menu - [Rendering]
You can save them by widely used picture format like BMP or JPG as rendering function is loaded.

Q10. Vertices, lines or faces are not shown

A10-1. [Pt][Ln][Fc] buttons are not pushed.
Please push [Pt][Ln][Fc] buttons above the each view.
A10-2. It displays the current object only.
Please confirm if the menu - View - Show only faces in a current object, or Show lines of uncurrent objects are checked.
A10-3. The visibility of the object is hidden in the object panel.
Please confirm that an icon of each object in the object panel is visible.
