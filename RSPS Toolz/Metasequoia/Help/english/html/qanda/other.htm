FAQ - Others

Q1. How to make objects into the image of such as BMP file

A1. Rendering
Save the rendering product in the appropriate image form in menu [File] [Rendering].

Q2. How to print

A2. Print function is not provided.
Metasequoia does not provide print function. Save the image in menu [File] [Rendering] and read and print it with other image editing softwares (like Paint standardly attached to Windows).
