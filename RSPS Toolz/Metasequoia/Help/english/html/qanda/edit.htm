Q1. It is difficult to select vertices or lines.

A1. The distance to select is small.
When you use the high resolving power, it sometimes be difficult to select vertices or lines. Increase the value of [Distance to select] in Configuration.

Q2. I cannot select vertices, lines, or faces with a click.

A2-1. Objects are prohibited to select.
If there is a key mark on the left side of the name column on object panel, that object is prohibited to edit. Click the key mark or remove the check of prohibiting edit in Object Property.
A2-2. [Sel Lock] mode is selected.
When [Sel Lock] is highlighted on the right side of the status bar, you cannot release selection. Switch ON and OFF in menu [Edit] [Lock selection] or with a space key on the keyboard.
A2-3. [CurObj] mode is selected.
When [CurObj] is ON in Edit option on command panel, you cannot edit objects except the current one. Make [CurObj] option OFF or activate the object which you want to select in object panel.

Q3. Shape of the object is broken in symmetric editing.

A3-1. A pair of the vertices does not exist.
A pair of the vertices is always necessary for a symmetric editing. A vertex without a pair is moved to X=0 and is fixed at the position if you set [Fix X=0] for [Move non-symmetry vertex] in Configuration, and it helps you to arrange it the symmetric center.
A3-2. Command is not available with symmetric editing.
All commands are not available with symmetric editing. Please confirm which command is available with the symmetric editing.

Q4. The area close to or remote from the viewpoint is not shown or shown in twisted shape.

A4-1. The problem of parameters of a camera
Twist caused by perspective occurs when 3D space objects are projected into 2D plane. If the distance from the camera to the target object becomes shorter in bigger view angle, twist seems to be conspicuous. So, change perspective and view angle in [View] command or select initialization in menu [View] [Set View].
A4-2. Use dolly
Assign [Dolly] for such as Wheel in Configuration. When you enlarge or reduce view, field of view is not changed and the distance from the view point changes in dolly.

Q5. Buttons on command panel are too small to press.

A5. Change style file
Select "DefLarge.style" in [Style] in Layout in Configuration.

Q6. Movement cannot be done towards Y axis on world coordinate.

A6. Press the Ctrl key
You can do movement towards Y axis with the Ctrl key.
It makes movement towards Z axis with the Ctrl key when you set [XY] in [Plane of world coordinate] in Configuration.

Q7. I clicked a mouse on area with nothing, but I cannot release the selection of vertices/lines/faces.

A7-1. [Joined Face] mode in [Select] command is selected.
When you click the selected object twice, the selection is released. Or release selection in menu [Edit] [unselect all] (Ctrl+D).
A7-2. [Sel Lock] mode is selected.
You cannot release the selection when [Sel Lock] is highlighted on the right side of the status bar. Switch ON and OFF in menu [Edit] [Lock selection] or the space key on the keyboard.

Q8. It is difficult to edit objects because the view rotates around the origin.

A8. Make the selected part to the center of the rotation of the viewpoint.
Select a vertex, a line, or a face near the position you want to make it center and select menu [Selected] [Set center of rotation (Ctrl+W). Then you can make that position to the center of the rotation of the viewpoint. And if you also select menu [Selected] [Fit to view] (Ctrl+F), it becomes convenient because the screen fits to the selected area.

Q9. Objects become twisted when I select patch and lathe.

A9. Because lathe applies to both sides.
It is a specification. Freeze lathe in menu [Object] [Freeze] and unify its face in menu [Object] [Unify faces], and apply a patch to the object.

Q10. I want to customize shortcut keys.

A10. Change in Configuration
You can customize by clicking [Customize shortcut keys] in 'Keyboard' in Configuration. But, this is only assigned to some specific buttons or keys. Operation of mouse buttons can be set up in 'Mouse' page in Configuration.

Q12. Middle button of mouse does not do anything

A12. Mouse driver or utility assigns other action
Some mouse driver or utility can assign other actions to mouse buttons, and middle button is not available in the condition. Please assign an original action to midlle button in the mouse utility or uninstall the mouse utility.
