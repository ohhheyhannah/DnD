選択

3D画面上で直接選択したい頂点・辺・面を指定します。

通常

通常の選択操作を行います。

Shiftキーを押しながら頂点・面をドラッグすると、連続的に選択・非選択できます。（辺に対しての選択は出来ません）

接続面

頂点・面を指定し、その面につながっているすべての面を選択・非選択します。（辺に対しての選択は出来ません）

ベルト

辺を指定して、その辺を持つ面からベルト状に頂点・面に対して選択・非選択を行います。面の選択オプションがOFFのときは辺が選択されます。

連続辺

クリックまたはドラッグした辺につながっている辺を連続的に選択します。

クリック時には最も遠くまでつながっている辺まで一括して選択し、ドラッグ時には開始位置の辺から徐々に範囲を広げながら選択します。

拡張

ドラッグ開始位置から、ドラッグする距離によって頂点・面を選択する範囲を徐々に広げていきます。

ルート

頂点・辺・面のいずれかを２か所以上クリックすると、最短経路上の頂点・辺・面が選択されます。右クリックまたはダブルクリックで選択を終了します。

現物指定

指定した頂点・辺・面のオブジェクトを現在のオブジェクトにします。

ブラシ選択

ドラッグによってマウスカーソル付近の頂点を選択します。
